# 💰 MiFinanzas - Control de Finanzas Personales

App web para controlar tus finanzas personales. Construida con Next.js, TypeScript, Tailwind CSS y shadcn/ui.

## ✨ Características

- **Dashboard** con resumen financiero, gráficos y estadísticas
- **Transacciones** — registro de ingresos y gastos con categorías
- **Metas de ahorro** con seguimiento de ítems (checklist)
- **Meta de Quinceañera** precargada con 10 ítems
- **Promedios mensuales** de ingresos y gastos
- **Gráficos** de los últimos 6 meses (Recharts)
- **100% en español** — interfaz intuitiva y responsive
- **Sin servidor** — los datos se guardan en tu navegador (localStorage)
